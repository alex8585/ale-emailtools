== Description ==
Плагин автоматической интеграции интернет-магазина на Woocommerce с сервисом брошенных корзин и триггерных рассылок [Emailtools](https://emailtools.ru)

Email-рассылки для забытых корзин, брошенных категори, брошенных товаров, брошенных категорий, брошенных оплат и прочих триггеров.

Реанимируем потенциальных покупателей, которые добавили товар в корзину, но не оформили заказ.

Возвращаем до 30% потерянной прибыли.

Список добавляемых функций:
* Триггер "Брошенная корзина" [Пример писма](https://emailtools.ru/blog/trigger-broshennaya-korzina)
* Триггер "Брошенная оплата
* Триггер "Брошенный просмотр категории" [Пример писма](https://emailtools.ru/help/zapusk-broshennoy-kategorii)
* Триггер "Брошенный просмотр товара" [Пример писма](https://emailtools.ru/help/zapusk-broshennogo-tovara)
* Зака оформлен
* Триггер "Пройдите опрос / оставьте отзыв" [Пример писма](https://emailtools.ru/blog/trigger-ostavte-otzyv)
* Триггер "Поделитесь ссылкой на сайт"

Исходный код плагина написан в высоком качестве для легкого добавления функций, а также для стабильности и высокой безопасности, проверен с службой WordPress.

Подробнее о триггерах [здесь](https://emailtools.ru/blog/triggery-emailtools)

Преимущества сервиса [Emailtools.ru](https://emailtools.ru)
Условия использования сервиса и дополнения на сайте [Emailtools](https://emailtools.ru)

Триггерные кампании могут быть использованы для реанимации заказов, поднятия среднего чека, повторных продаж. Работают для продажи как товаров, так и услуг и увеличивают продажи в комплексе с email-рассылкой в среднем на 30%.

Сервис запоминает всех пользователей сайта когда либо оставивших свой email на сайте в любой форме и не требует их повторной регистрации или авторизации. А также, Emailtools.ru может узнать пользователей которые попали в вашу базу email до его установки.

При регистрации на сайте вы можете получить бонусные письма на баланс личного кабинета, что обеспечит вам бесплатный пробный период. А также ознакомиться с результатами работы сервиса на примере уже подключённых интернет-магазинов.

Сервис создан специально для владельцев небольших и средних интернет-магазинов, потому что был создан на основе технологии маркетинговой системы, что обеспечивает высокую эффективность работы и при этом его система тарификации ориентирована на малый бизнес и реализована на одной из наиболее устойчивых и распространённых систем Unisender (Unione).

В [блоге](https://emailtools.ru/blog) вы найдёте описание функционала сервиса, кейсы и примеры эффективных кампаний, а также [бесплатное руководство](https://emailtools.ru/help) по настройкам и видеоинструкции.

Установка плагина: 
1. Зайдите в меню Плагины-Добавить новый, найдите "emailtools", нажмите установить
2. Активируйте плагин 
3. Перейдите в Настройки-Emailtools setting, добавьте API Key (EMT API KEY), на время тестирования поставьте флажок "Keep a log file" и "Terms and conditions" и нажмите кнопку Save Changes. После тестирования уберите флажок с пункта "Keep a log file". Логирование будет доступно по ссылке ВашДомен/emtlog.txt
4. API Key (EMT API KEY) находится в личном кабинете [по этой ссылке](https://emailtools.ru/settings). Зарегистрируйтесь если у вас ещё нет личного кабинета на Emailtools.ru, это быстро и просто - [регистрация в Emailtools](https://emailtools.ru/registratsiya)
